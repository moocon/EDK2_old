Welcome to the home of EDK 2.0 Development!

If you are new to the project, welcome! We are still in heavy development, but we would invite you to take a look at what we have done so far.

Your feedback is critical to making EDK 2.0 a success! Please submit any enhancements, defects, or requests through the Project Tracker tool.

Getting Started

To get the code please follow this document. It will walk you through the installation of the Subversion Software Configuration Management Client, TortoiseSVN, and show you how to pull the latest copy of the tree.

Install some external tools that are necessary to build the code.

Further instructions for using the EDK2.0 are provided in the ReleaseNotes.txt at the top of the edk2 tree. You may also view the Release Notes using the web interface to the Subversion repository.

If you have questions about the code, please post a note on the dev@edk2.tianocore.org mailing list. One of our friendly developers will be happy to help you. You may subscribe to the mail lists of the edk2 project here.
