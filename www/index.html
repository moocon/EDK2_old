Welcome to the home of EDK 2.0 Development!

If you are new to the project, welcome! We are still in heavy development, but we would invite you to take a look at what we have done so far.

Your feedback is critical to making EDK 2.0 a success! Please submit any enhancements, defects, or requests through the Project Tracker tool.

Getting Started

To get the code please follow this document.

There are several tools that you will need to install on your machine in order to compile the EDK 2.0 code. Instructions for obtaining and installing these tools are provided in the ReleaseNotes.txt at the top of the edk2 tree.

It is necessary to read through the release notes in order to find out what you can do with the EDK 2.0 at this stage. You will find them at the top of the tree that you get from our source repository.

If you have questions about the code, please post a note on the dev@edk2.tianocore.org mailing list. One of our friendly developers will be happy to help you.
